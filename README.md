# puts-univ

장기발전계획 · 대학기관평가인증 · 대학혁신지원사업 통합 성과관리 화면 프로토타입.

교육혁신처가 실적을 한 번 입력하면 세 자료에서 함께 활용할 수 있다는 것을 보여주기 위한 화면 시안입니다.
모든 데이터는 `src/lib/mock-data.ts`의 더미 데이터이며, 실제 DB·로그인 연동은 포함되어 있지 않습니다.

## 실행

```bash
npm install
npm run dev
```

## 페이지

- `/` 통합 현황
- `/long-term-plan` 장기발전계획 (2023년~)
- `/evaluation` 대학기관평가인증 (2023년~)
- `/uisp` 대학혁신지원사업 (2025년~, uisp.vercel.app 용어 기준)
- `/data-mapping` 교육혁신처 데이터 연계 (대학·신대원·대학원 소속별 분포 포함)
- `/documents` 문서형 보기
